# Freeze Milestone 0 exact semantics and fixtures (RLT-003)

Date: 2026-07-12

Status: integrated

Relevant commit(s): `8bc88d53aa57092adcac336c3ca74f0f6bea3638`

## Problem / why this was next

Fee meaning, rounding, amount encoding, reserve transitions, snapshot identity, candidate validity, and deterministic ties needed an accepted contract before any financial implementation could be trusted.

## Decision or implementation

Keep exact financial values and intermediates as bigint, represent the charged fee fraction explicitly, apply one final integer floor to the combined constant-product formula, credit gross input to reserves, bind plans to immutable snapshot ID and checksum, and require exact replay before incumbent acceptance. Six small fixtures were derived independently of future production helpers.
